Bank Account

Languages: .NET, C#

This is a console application that allows the user to check account balances for checking and savings accounts, withdraw and deposit money, and view client information.

Fortune Teller

Languages: .NET, C#

Fortune teller is a console application that tells the user's fortune by asking a few questions such as age and birth month.

Lucky Numbers

Languages: .NET, C#

The lucky numbers console application generates a set of random numbers in a range specified by the user, asks the user for their guesses, and then compares user guesses to the random generated numbers. A prize is then awarded based on the number of matching numbers in both sets.

Tuscany Villa

Languages: HTML, CSS

This is a mock-up created for a website that features rental properties in Tuscany.

Virtual Pet

Languages: .NET, C#

This application allows the user to check on the status of their pet and interact with it. The pet's status changes based upon user action. For example, the pet's cage becomes dirty if the pet is fed.

Virtual Pet Shelter

Languages: .NET, C#

The application prompts the user to enter whether they are a manager or shelter volunteer. From there, the choices available to the user are dependent upon that status. Volunteers can interact with the pets, feed them, etc., while managers can view employee information, check on the status of the pets, and initiate the adoption process.
